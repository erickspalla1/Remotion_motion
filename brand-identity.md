# Brand identity

Brand identity is a visual system, not a logo. A logo is one element in a system that includes typography, color, graphic language, photography style, tone of voice, and how all of these behave together across every touchpoint.

## Brand creation process

### Phase 1: Brand strategy (before any visuals)

Understand these before designing anything:

- **Positioning:** what space does this brand own in the audience's mind?
- **Values:** what does the brand believe? (3-5 core values, not a list of 20)
- **Personality:** if the brand were a person, how would they speak and dress?
- **Audience:** who specifically? Demographics AND psychographics.
- **Competitive landscape:** what do competitors look like? Where is the visual white space?

### Phase 2: Visual direction

Translate strategy into visual decisions:

1. **Choose the visual style** — reference `visual-styles.md`. The style must match the brand personality.
2. **Define the type system** — reference `typography.md`. Primary, secondary, and accent typefaces.
3. **Build the color system** — reference `color.md`. Full palette with roles defined.
4. **Define the graphic language** — what shapes, patterns, textures, and graphic devices make this brand recognizable beyond the logo?

### Phase 3: Core assets

Build these in order:

1. **Wordmark or logomark** — the primary brand mark
2. **Type system** — applied to real content (headings, body, captions)
3. **Color system** — applied to real surfaces (cards, backgrounds, buttons)
4. **Graphic elements** — patterns, shapes, dividers, frames that carry the brand
5. **Photography/illustration direction** — what kind of imagery, what treatment
6. **Application examples** — the brand on real surfaces (social post, business card, website header)

### Phase 4: Brand guidelines document

Codify everything into a reference document that any designer can follow:

- Logo usage (clear space, minimum size, do's and don'ts)
- Color palette (primary, secondary, accent, functional, with hex/RGB values)
- Typography (font names, weights, size scale, pairing rules)
- Graphic elements (patterns, shapes, when and how to use)
- Photography direction (style, treatment, subjects, mood)
- Tone of voice (writing style examples)
- Application examples (mockups showing correct usage)

## Logo principles

### Types of logos

| Type | What it is | Best for | Examples |
|---|---|---|---|
| Wordmark | Brand name in custom type | Distinctive names, text-focused brands | Google, Coca-Cola, FedEx |
| Lettermark | Initials | Long names, B2B, established brands | IBM, HBO, CNN |
| Symbol / icon | Abstract or pictorial mark | Global brands, app-first, needs to work at 16px | Apple, Nike, Twitter |
| Combination | Symbol + wordmark | New brands that need both recognition paths | Spotify, Airbnb, Mastercard |
| Emblem | Text inside a shape | Heritage, institutions, badges | Starbucks, Harvard, NFL |

**Default recommendation for new brands:** combination mark. It gives maximum flexibility — use the full lockup when there is space, symbol-only when there is not.

### Logo design principles

1. **Works in one color** — if the logo loses its meaning in black on white, the color is a crutch
2. **Works at 16px** — social media favicons, app icons, browser tabs. Must be recognizable
3. **Works at billboard scale** — no details that disappear at distance or become overwhelming up close
4. **Distinctive silhouette** — cover the logo with your hand; the outline alone should be recognizable
5. **No trends** — logos last 10-20 years. Gradients, shadows, and 3D effects date quickly. Flat and simple endures
6. **Appropriate, not decorative** — the logo communicates what the brand is, not what the designer thinks is cool

### Logo spacing

Define a clear space zone around the logo equal to the height of a key element (like the cap-height of the wordmark or the radius of the symbol). Nothing enters this zone.

## Brand consistency vs flexibility

A rigid brand dies. A brand with no rules is not a brand.

The solution is a **flexible system with clear boundaries:**

**Fixed elements** (never change):
- Logo (exact proportions, colors, clear space)
- Primary typeface
- Core color palette values

**Flexible elements** (vary within rules):
- Secondary type choices (can vary by medium)
- Color application (which color dominates varies by context)
- Graphic element density (minimal for luxury context, rich for campaign)
- Photography style (subject matter varies, treatment stays consistent)

**Seasonal/campaign layers** (temporary additions):
- Campaign-specific color accents
- Campaign typography additions
- Temporary graphic motifs
- These sit ON TOP of the base brand, never replace it

## Brand for motion

A motion brand system includes everything above plus:

- **Motion personality** — the default speed, easing, and energy level (see `motion-direction.md`)
- **Signature transition** — one transition that is "ours" (example: Spotify's color wipe, Apple's slow zoom)
- **Logo animation** — how the logo enters and exits (this single animation is seen thousands of times; it must be impeccable)
- **Sound identity** — audio logo or sonic signature
- **Timing standards** — default durations for common formats

## Testing a brand identity

Before delivering, test the brand across these scenarios:

1. **Squint test** — at arm's length, is the brand recognizable by color + shape alone?
2. **Thumbnail test** — at 32x32 pixels, does the mark still read?
3. **Context test** — place the brand next to competitors. Does it stand out or blend in?
4. **Misuse test** — can someone easily use the brand incorrectly? If yes, add guardrails to the guidelines
5. **Time test** — will this look dated in 3 years? In 10 years? Strip anything that will
6. **Medium test** — does it work on a white background? Dark? Over photography? On a video? On a tote bag?
